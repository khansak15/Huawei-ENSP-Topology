Modern enterprise networks demand a sophisticated integration of heterogeneous technologies 
operating across multiple layers of the OSI model to deliver reliable, secure, and scalable 
communication infrastructure. This project addresses the Complex Computing Problem (CCP) for the 
Computer Communication Networks course (CT-376, Spring 2026) by designing, simulating, and 
documenting a comprehensive network topology that unifies the following core technologies: 

  ● IP Addressing: Subnetting with Variable Length Subnet Masks (VLSM) utilising /30 
  point-to-point links and /24 LAN segments. 
  
  ● Switching: Virtual Local Area Networks (VLANs) with Router-on-a-Stick (ROAS) inter-VLAN 
  routing, Spanning Tree Protocol (STP) for Layer 2 loop prevention, and Eth-Trunk (LACP) for link 
  aggregation. 
  
  ● Routing: A combination of Static Routing for deterministic path control and Dynamic Routing 
  via the Routing Information Protocol version 2 (RIP v2) for automatic route propagation across 
  a multi-router backbone. 
  
  ● Management & Security: Telnet-based remote management with AAA authentication, and 
  Basic Access Control Lists (ACLs) applied to VTY lines for telnet access restriction on the core 
  router.
  
  ● Application Layer Services: File Transfer Protocol (FTP) server deployment for centralised file 
  distribution.
